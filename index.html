Python PSQL

This is one of my favorite projects because I found out I really, really like organizing and reading SQL data. I was given an SQL file that contained 11.9mb of data with the task of answering three questions with it. I had to answer these multiple part questions by using a single query. I figured out how many steps it would take to answer different parts of the same question, created all the VIEWs needed for them, and then switched on over to Python. In Python, I wrote the functions that would print out the query results and had it each question answer print out in an easy to read table.
